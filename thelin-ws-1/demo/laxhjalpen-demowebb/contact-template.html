Demo of contact form for WaSP Interact

This is an example of a template to be used with the contact.php script for form handling.

A student may read this source code in order to see how to put PHP generated data into the form and what values name attributes are supposed to have. Class names, order of elements, etc, may be altered. However, attribute names and PHP-variable names must remain the same. The PHP script will automatically add selected and/or checked attributes to the checkbox and radiobuttons, should the user wish to keep them.

This is a demo only. Actual usage of this at this location is disabled. No recipient will read your messages.
