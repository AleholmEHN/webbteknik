Editor

  • Textredigerare
  • Texteditor
  • Ett program som hanterar rå text
  • Inte Word, Libre Office Writer, etc
  • Raw text är inte rich text

Grundkrav på en editor

  • Stöd för olika teckenkodningar
  • Radnumrering
  • Syntax highlighting

Anteckningar

  • Stödjer numera UTF-8, men saknar bra inställningar för standardvärdet
  • Radnumrering? Ja, det också numera.
  • Syntax highlight? Nix!
  • Många andra saker…
