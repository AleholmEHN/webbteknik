Webkits ursprungliga förslag


-webkit-gradient(
    linear,         /* Typ */
    top left,       /* Start */
    bottom left,    /* Slut */
    color-stop(),
    color-stop()
)

Nackdelar

  • Mångordig (verbose)
  • Invecklad, krånglig syntax

Förenklad syntax i Firefox


-moz-linear-gradient()
-moz-radial-gradient()
-moz-repeating-linear-gradient()
-moz-repeating-radial-gradient()

Stöd för denna syntax

  • Firefox (-moz-)
  • Safari, Chrome, iOS, Android, etc (-webkit-)
  • Opera 12 (-o-)
  • Internet Explorer 10 (-ms-) – Onödig!

Problem med denna syntax

  • Inte konskevent med annan CSS (vinklar)
  • Utrymme fanns för ytterligare förenkling

Den slutliga W3C-syntaxen

  • Firefox 16 (sommaren 2012)
  • Opera 12.1 (sensommar 2012)
  • Internet Explorer 10 (senhöst 2012)
  • Men ännu inte i Webkit (sägs oktober 2012)
